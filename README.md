Este proyecto contiene los ejercicios de modularizacion en Java. El codigo a sido guardado en metodos independientes para mejorar la lectura, el mantenimiento y utilizar el software.

.public: el metodo es accesible desde cualquier lado.
.Static: permite que el programa ejecute el metodo sin tener que crear un objeto.
.Void: realiza una tarea pero no la devuelve un resultado matematico al final
. main: es el nombre reservado en java como inicio de todo
